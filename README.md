## Intro
### 목표 설정부터 일기 작성, 교환일기까지 한 번에 관리할 수 있는 목표 관리 SNS 일기 플랫폼입니다.
1. 사용자는 스스로 목표를 정하고 매일 성공 여부를 기록하며, 그룹을 통해 서로의 목표 진행 상황을 응원하고 공유할 수 있습니다.
2. OpenAI 기반 감정 이모지 요약 기능을 통해 사용자는 글을 쓰는 것만으로도 감정과 성과를 직관적으로 시각화할 수 있습니다.
